# Health – iOS Health Tracking Application

Health is a modern iOS application designed to help users monitor and improve their well-being through an intuitive and user-friendly interface. The application centralizes personal health information, allowing users to record medical indicators, track their health history, receive personalized recommendations, and manage medication reminders in a single place.

Developed as a Bachelor's thesis in Software Engineering, the project demonstrates the design and implementation of a health monitoring system following software engineering best practices, including requirements analysis, UML modeling, MVC architecture, database design, and mobile application development for iOS.

## Key Features

- 👤 Secure user registration and authentication
- ❤️ Health metrics tracking (blood pressure, heart rate, blood glucose, etc.)
- 📊 Health history and progress monitoring
- 💡 Personalized health recommendations
- 💊 Medication and appointment reminders
- 📈 Health reports and data visualization
- ⚙️ User profile management
- 🔒 Secure storage of personal medical information

## Technology Stack

- **Platform:** iOS
- **Language:** Swift
- **Architecture:** MVC (Model–View–Controller)
- **Database:** Core Data / Realm (project design)
- **Development Environment:** Xcode

## Project Goal

The primary goal of this project is to provide users with a convenient, centralized solution for monitoring their health while promoting healthy lifestyle habits through personalized insights, health analytics, and timely reminders.
